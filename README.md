# فروشگاه آنلاین تکنولایف

پروژه فروشگاه آنلاین با React  (App Router)

## قابلیت‌ها
- صفحه اصلی با اسلایدر
- فیلتر برند و جستجو
- سبد خرید با Context API
- صفحه جزئیات محصول

## تکنولوژی‌ها
- React.js
- Tailwind CSS
- Context API
- json-server (برای شبیه‌سازی بک‌اند)

## نصب و اجرا
```bash
npm install
npm run dev
# در ترمینال دیگر:
npx json-server --watch db.json --port 3000  یا npm run server
